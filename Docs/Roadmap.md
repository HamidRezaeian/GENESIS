# GENESIS Roadmap

This roadmap outlines the past, present, and future phases of the GENESIS Digital Universe project.

---

## 🟢 Phase 1: The Thermodynamic Filter (Completed)
**Goal:** Prove that energy constraints drive self-organization.
- [x] Basic Node/Edge graph physics.
- [x] Metabolism and edge maintenance costs.
- [x] Emergence of stable, sparse topologies from chaos.
- [x] Free Energy Principle: Tying survival to predictive accuracy.

## 🟢 Phase 2: Darwinian Neuroevolution (Completed)
**Goal:** Enable reproduction, mutation, and the growth of complex brains.
- [x] Asexual reproduction mechanics.
- [x] Heredity of traits (Learning Rate, Reproduction Threshold).
- [x] Coevolution: Builders vs. Exploiters (The Arms Race).
- [x] Structural Neuroevolution: The `DynamicBrain` (NEAT-like).
- [x] The Catastrophe Mechanic to prevent overfitting.

## 🟢 Phase 3: Language and Differentiable Communication (Completed)
**Goal:** Solve the Symbol Grounding problem.
- [x] Implementation of hidden environmental hazards.
- [x] Peer-to-peer signaling channels.
- [x] **Multi-Agent Backpropagation:** Passing gradients between separate agents without supervision.
- [x] Emergence of a unified communication protocol (> 95% accuracy).

## 🟡 Phase 4: UI / UX Overhaul (Completed)
**Goal:** Transform the raw physics engine into an interactive Observation Deck.
- [x] High-performance HTML5 Canvas rendering.
- [x] Live Leaderboard of smartest agents.
- [x] "Follow Camera" for tracking subjects.
- [x] Golden Auras to highlight evolutionary winners.

## 🔵 Phase 5: Spatial Physics & True Multi-Cellular Life (Planned)
**Goal:** Move from abstract graph topologies to a Euclidean 2D space.
- [ ] Implement X/Y coordinate physics (Movement, Velocity, Mass).
- [ ] Vision Sensors: Allow agents to see pixels/entities in a radius instead of abstract neighbors.
- [ ] Physical clustering: Allow nodes to physically bind together to form multi-cellular organisms with specialized parts (Sensors, Motors, Brains).

## 🔵 Phase 6: Memory & Statefulness (Planned)
**Goal:** Evolve Recurrent Neural Networks (RNNs).
- [ ] Allow `DynamicBrain` to form cyclic edges (memory loops).
- [ ] Test if agents can remember past events to solve temporally delayed hazards.
