# Product Requirements Document (PRD)

## 1. Project Overview
**Name:** GENESIS Digital Universe
**Vision:** To create a digital environment where Artificial General Intelligence (AGI) emerges organically through the Free Energy Principle, thermodynamic physics, and evolutionary pressure, rather than through explicitly programmed biological rules.
**Tagline:** Intelligence from Thermodynamics.

## 2. Core Philosophy
Intelligence is the inevitable byproduct of a system trying to minimize surprise (predictive error) and conserve energy (thermodynamic cost) in a hostile environment. GENESIS tests this philosophy by simulating an artificial ecosystem that enforces strict mathematical and informational rules on energy, forcing agents to evolve highly efficient, computationally cheap algorithms (Opcodes) to survive.

## 3. Target Capabilities
1. **Self-Organization:** Random streams of code must form stable, replicating loops.
2. **Computational Efficiency:** Organisms must naturally select for minimum CPU cycles to solve environmental tasks.
3. **Darwinian Evolution:** The system must naturally select for traits through heredity, mutation, and survival of the fittest.
4. **Red Queen Arms Race:** Parasitic exploiters and builders must coevolve to prevent the system from stagnating.
5. **Structural Neuroevolution:** Agents must organically grow their logic capacity (Turing Completeness) only when the environment justifies the thermodynamic cost.
6. **Emergent AGI:** Agents must invent algorithmic approaches to environmental logic problems to survive, mirroring the human brain's 20W efficiency.

## 4. User Experience & Dashboard
The frontend must provide an "Observation Deck" for the user to watch evolution unfold:
- **Real-Time Visualizer:** A high-performance HTML5 Canvas rendering the RAM/Memory space in real-time.
- **Live Leaderboard:** Tracking the most successful execution threads (Total Replications and Avg Energy).
- **Golden Auras:** Elite/smart threads (Alphas) must emit glowing halos for easy identification.
- **Live Tuning & Controls:** Play, Pause, Reboot controls.

## 5. Success Metrics
- **Stability:** The population should reach a dynamic equilibrium (neither exploding nor going extinct).
- **Learning Convergence:** Prediction errors should drop rapidly over generations.
- **Language Success Rate:** Agents must cross the Symbol Grounding threshold with > 95% accuracy in communication tasks.
