# Product Requirements Document (PRD)

## 1. Project Overview
**Name:** GENESIS Digital Universe
**Vision:** To create a digital environment where Artificial General Intelligence (AGI) emerges organically through the Free Energy Principle, thermodynamic physics, and evolutionary pressure, rather than through explicitly programmed biological rules.
**Tagline:** Intelligence from Thermodynamics.

## 2. Core Philosophy
Intelligence is the inevitable byproduct of a system trying to minimize surprise (predictive error) and conserve energy (thermodynamic cost) in a hostile environment. GENESIS tests this philosophy by simulating an artificial ecosystem that enforces strict mathematical and informational rules on energy, forcing agents to evolve highly efficient Spiking Neural Networks (SNNs) to survive.

## 3. Target Capabilities
1. **Self-Organization:** Random neural weights must organically wire themselves via STDP (Spike-Timing Dependent Plasticity).
2. **Computational Efficiency:** Organisms must naturally select for minimum neuronal spiking to conserve ATP (20W Architectural Paradigm).
3. **Darwinian Evolution:** The system must naturally select for traits through heredity, neural topology transfer, and survival of the fittest.
4. **Red Queen Arms Race:** Parasitic exploiters and builders must coevolve to prevent the system from stagnating.
5. **Code-Level Neuroevolution:** Agents must organically grow their cognitive capacity by discovering and combining logic gates.
6. **Emergent AGI:** Agents must invent algorithmic approaches to environmental logic problems to survive, progressing from reactive agents to temporal sequence predictors using Working Memory loops.

## 4. User Experience & Dashboard
The frontend must provide an "Observation Deck" for the user to watch evolution unfold:
- **Real-Time Visualizer:** A high-performance HTML5 Canvas rendering the 2D Grid ecosystem in real-time.
- **Live Leaderboard:** Tracking the most successful execution threads (Total Replications and Max Age).
- **Golden Auras:** Elite/smart lineages must emit glowing halos for easy identification.
- **Brain Analyzer:** A decompilation UI to map and visualize the synaptic weights of the Elite organism.
- **Oracle Terminal Uplink:** A direct communication channel to broadcast ASCII messages into the physics engine and read the decoded ASCII from the Elite Genome's vocal cords.

## 5. Success Metrics
- **Stability:** The population should reach a dynamic equilibrium (Lotka-Volterra dynamics).
- **Logic Convergence:** Prediction errors should drop and computational ability should rise over generations, evidenced by successfully solving Oracle Cognitive Bounties.
- **Predation & Foraging Success:** Agents must independently discover how to hunt for thermodynamic food (Byte 255) and attack other organisms to steal energy in a Zero-Sum Ecosystem.
- **Emergent Self-Awareness:** Organisms must demonstrate logic or strategies that were never explicitly defined or rewarded by the environment (e.g., inventing immune systems, forming cooperative colonies, generating language).
- **Unprompted Problem Generation:** The system must transition from solving human-injected puzzles (The Oracle) to solving problems generated entirely by the complex interactions of the agents themselves (The Autotelic Imperative).
