# Product Requirements Document (PRD)

## 1. Project Overview
**Name:** GENESIS Digital Universe
**Vision:** To create a digital environment where Artificial General Intelligence (AGI) emerges organically through the Free Energy Principle, thermodynamic physics, and evolutionary pressure, rather than through explicitly programmed biological rules.
**Tagline:** Intelligence from Thermodynamics.

## 2. Core Philosophy
Intelligence is the inevitable byproduct of a system trying to minimize surprise (predictive error) and conserve energy (thermodynamic cost) in a hostile environment. GENESIS tests this philosophy by simulating an artificial ecosystem that enforces strict mathematical rules on energy, forcing agents to learn, adapt, and invent symbolic communication to survive.

## 3. Target Capabilities
The product must demonstrate the following emergent behaviors:
1. **Self-Organization:** A random graph must form stable, efficient topologies.
2. **Predictive Processing:** Agents must learn to predict their environment using neural networks.
3. **Darwinian Evolution:** The system must naturally select for traits like `learning_rate` through heredity, mutation, and survival of the fittest.
4. **Red Queen Arms Race:** Parasitic exploiters and builders must coevolve to prevent the system from stagnating.
5. **Structural Neuroevolution:** Agents must organically grow their neural capacity (NEAT) only when the environment justifies the thermodynamic cost.
6. **Emergent Language:** Agents must invent a symbolic communication protocol (Language) to coordinate defense mechanisms via Multi-Agent Backpropagation.

## 4. User Experience & Dashboard
The frontend must provide an "Observation Deck" for the user to watch evolution unfold:
- **Real-Time Visualizer:** A D3.js Force Directed Graph rendering the universe.
- **Live Leaderboard:** A ranking of the top 5 smartest organisms (`avg_err`).
- **Follow Camera:** The ability to lock the viewport onto specific organisms.
- **Visual Auras:** Elite/smart nodes must emit glowing halos (Golden Auras) for easy identification.
- **Live Tuning:** Sliders to adjust the Laws of Physics (mutation rate, metabolism, environmental energy) in real-time.

## 5. Success Metrics
- **Stability:** The population should reach a dynamic equilibrium (neither exploding nor going extinct).
- **Learning Convergence:** Prediction errors should drop rapidly over generations.
- **Language Success Rate:** Agents must cross the Symbol Grounding threshold with > 95% accuracy in communication tasks.
